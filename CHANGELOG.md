# Changelog

All notable changes to this project are documented here. Until the first
stable release, additions may refine APIs while preserving the documented
`0.1` behavior whenever practical.

## 0.1.0

- Added row-oriented and column-oriented table schemas.
- Added reusable field parsers and annotation-driven conversion.
- Added source-aware diagnostics, record validation, and table validation.
- Added project-owned cell DSL rules and table transformations.
- Added grouped column expansion with replaceable range and repeat rules.
- Added local record references and output model adapters.
- Added reusable field components and discriminated record variants.
- Added default factories, aliases, parsing policies, and error collection.
- Added schema introspection, transformer composition, and cell DSL composition.
- Added static Gherkin table checking and the `bdd-tablex check` command.
